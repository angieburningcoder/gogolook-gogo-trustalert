## 🎯 快速導航

### 我是新手，想快速上手
1. 先看 [快速開始指南](./development/QUICKSTART.md)（5 分鐘）
2. 測試本地開發環境
3. 閱讀 [快速參考手冊](./development/CHEATSHEET.md) 了解常用操作

### 我要部署到生產環境
1. 閱讀 [部署指南](./deployment/DEPLOYMENT.md)
2. 設定環境變數
3. 部署到 Vercel

### 我想了解產品驗證策略
1. 閱讀 [產品洞察與測試策略](./product/INSIGHTS.md)
2. 設定 Analytics 追蹤
3. 開始收集數據並分析

### 我想深入了解技術架構
1. 閱讀 [技術架構文檔](./development/ARCHITECTURE.md)
2. 查看 [專案結構](./development/PROJECT_STRUCTURE.txt)
3. 查看根目錄的主 [README.md](../README.md)
